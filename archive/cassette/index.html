Blokus Duo: Using Deep Search Algorithms to Explore Turn-Based Game Strategy

My PGSS culminating team research project. We completed extensive research to determine the best way to computationally approach strategic gameplay in the two player BlokusDuo board game. This project involved creating a framework and UI for the game as well as minimax, reinforcement learning, and Monte Carlo tree search agents to compete with humans.
